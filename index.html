Sekarang saatnya push, pertama tulis git init (digunakan dalam meinisialisasi github kedalam folder tersebut) git add (menambahkan semua file yang akan dimasukkan kedalam keranjang dan siap di upload) git commit -m “First Commit”(Fungsi ini digunakan upload untuk pertama kalinya) 10. copy github git remote add origin https://github.com/desipradi/Latihan-Github.git 11. lalu upload dengan ketik git push origin master 12. cek github jika sudah muncul file index.html berarti file sudah selesai upload dan cek file tersebut 13. selesai
